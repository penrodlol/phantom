## Label

<Note type="tip" title="Accessibility Tip">
  It is recommend to use a label as it describes the purpose of the switch to the user and is read
  by screen readers.
  <br />
  If you don't want to display a label, you can use the `aria-label{:inline}` attribute to provide a
  label. A common use case for this could be on a dark/light mode switch.
</Note>
